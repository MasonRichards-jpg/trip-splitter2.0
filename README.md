# Trip Splitter

A web application designed to simplify expense management and cost splitting among travel companions. Trip Splitter helps groups track shared expenses during trips and automatically calculates who owes whom.

**[Visit the Application](https://trip-splitter2-0.onrender.com/trip)**

## Project Description

Trip Splitter is an expense tracking and settlement application built for travelers who need to manage shared costs during group trips. Whether you're planning a vacation with friends, a family getaway, or a road trip, this application streamlines the process of recording expenses and determining fair settlements between participants.

The application provides an intuitive interface for adding group members, logging expenses, and automatically calculating the final settlement amounts to ensure everyone pays their fair share.

## Key Features

- **Group Management** - Create and manage travel groups with multiple participants
- **Expense Tracking** - Log shared expenses with detailed descriptions, amounts, and payers
- **Automatic Calculations** - Intelligent algorithm calculates who owes whom and settlement amounts
- **User-Friendly Interface** - Clean, responsive design for easy access on any device
- **Real-Time Updates** - See expense summaries and settlement calculations instantly
- **Cost Distribution** - Split expenses equally, by percentage, or with custom amounts
- **Settlement Summaries** - Clear breakdown of all payments owed between group members

## Technology Stack

- **Frontend**: HTML (48.9%), CSS (20.8%), JavaScript (4.3%)
- **Backend**: Python (26%)
- **Deployment**: Render

Visit the live application: [Trip Splitter](https://trip-splitter2-0.onrender.com/trip)
